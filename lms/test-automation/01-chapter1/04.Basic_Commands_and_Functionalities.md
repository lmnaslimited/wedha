## Basic Commands

### 1. **cy.visit()**

 **Functionality**: Loads a remote URL into the Cypress Test Runner.
 ```javascript
 cy.visit('https://example.com')
 ```

### 2. **cy.get()**
   **Functionality**: Queries for an element in the DOM using a CSS selector.
   ```javascript
cy.get('button').click()
 ```
